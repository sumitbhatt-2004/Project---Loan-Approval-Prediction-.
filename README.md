# 🏦 Loan Approval Prediction System

### 📊 Project Overview
This project is a Machine Learning solution designed to automate the loan eligibility process. By analyzing applicant data such as credit history, income, and education, the model predicts whether a loan application should be **Approved** or **Rejected**. This helps financial institutions reduce manual workload and minimize the risk of defaults.

### 🚀 Key Features
* **Data-Driven Insights:** Identifies high-risk applicants based on historical trends.
* **XGBoost Classifier:** Uses a powerful gradient boosting algorithm for high accuracy.
* **Hyperparameter Tuning:** Optimized using **GridSearchCV** to find the best model settings.
* **Data Preprocessing:** Handles missing values, outliers, and categorical encoding.

### 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, XGBoost, Matplotlib, Seaborn
* **Tools:** Kaggle, GitHub, Google Colab

### 📂 Dataset Description
The model analyzes several key factors:
* **Credit History:** Past credit reliability (Most important feature).
* **Applicant Income:** Total monthly income.
* **Loan Amount:** Total amount requested by the user.
* **Education:** Graduate or Not Graduate.
* **Property Area:** Urban, Semi-Urban, or Rural location.

### 📈 Model Performance
* **Model Used:** Optimized XGBoost
* **Evaluation:** Accuracy, Precision, Recall, and F1-Score (Calculated via GridSearchCV)
